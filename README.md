  本项目面向银行场景的多角色权限管理系统，采用前后端分离架构，实现以下核心功能

1.基于 JWT 令牌与拦截器实现角色权限动态校验，通过 URI 路径匹 配与静态资源目录隔离构建多角色访问控制，实现管理员、银行
职员、普通用户三级权限体系，差异化功能权限与操作界面；

2.支持开户、开卡、存取款、跨账户转账、客服反馈等核心银行业 务流程；

3.基于 RAG 架构，集成 DeepSeek 与 ChromaDB 向量数据库，构建具备银行知识检索能力的智能 AI 客服，并了结合 Vue 动态
渲染与 Element-Plus 交互组件，实现自然语言交互的账户查询、业务引导与问题解答。

  本项目成功建立起管理员、职员、用户三级权限体系，权限校验准确率达 100%，未发生越权访问事件；不同角色登录后系统界面与
功能菜单差异化管理，用户体验显著提升；AI 客服模块成功上线，能够有效处理账户查询、业务办理指引和常见问题解答。
